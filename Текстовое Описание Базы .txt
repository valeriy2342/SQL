База данных отеля - 'Hotel'
В база 9 сущностей - таблиц

1. Номера - 'rooms'
- id
- номер комнаты - совпадает с id
- категория номера
- вместимость номера
- цена номера

2. Посетители - 'users'
- id
- имя
- фамилия
- email
- телефон
- пол
- дата рождения
- rooms_id(ключ ссылающийся на основную таблицу rooms, на id номера)
- город по прописке


3. Бронирование - 'booking'(будущие)
- id
- id посетителя из таблицы users
- дата заселения
- дата выезда

4. Посетители - 'visitors'(база прошлых посетителей)
- id
- id посетителя из таблицы users
- дата заселения
- дата выезда

5. Фотоальбомы отеля - 'photo_albums_hotel'
- id
- имя альбома
- rooms_id(ключ ссылающийся на основную таблицу rooms, на id номера)

6. Фотографии комнат - 'rooms_photos'
- id
- album_id(ключ ссылающийся на таблицу photo_albums_hotel, на id фотоальбома)
- rooms_id(ключ ссылающийся на основную таблицу rooms, на id номера)

6. Отзывы об отеле - 'reviews' 
- id
- текст отзыва
- rooms_id(ключ ссылающийся на основную таблицу rooms, на id номера)
- users_id(ключ ссылающийся id посетителя из таблицы users)
- Время создания отзыва
- Имя - Данные оставившего отзыв если он не заселялся в отель 
- Фамилия - Данные оставившего отзыв если он не заселялся в отель 
- Телефон - Данные оставившего отзыв если он не заселялся в отель 

7. Виды Питание номера - 'nutrition' 
- id
- Категория питания номера
- users_id(ключ ссылающийся id посетителя из таблицы users)
- Время начала пакета питания
- Время окончания пакета питания

8. Акции(Скидки) - 'stoсks'
- id
- Размер скидки
- rooms_id(ключ ссылающийся на основную таблицу rooms, на id номера)
- Время начала акции
- Время окончания акции
